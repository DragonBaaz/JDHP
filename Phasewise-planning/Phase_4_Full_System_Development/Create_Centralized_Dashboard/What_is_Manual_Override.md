# What is Manual Override?

Manual override refers to the ability to intervene and control workflows manually when necessary. This ensures that the JDHP publishing house can handle unexpected situations effectively.

## Key Features:
1. **Error Handling**: Resolve issues that automated systems cannot handle.
2. **Task Control**: Start, stop, or modify tasks manually.
3. **Flexibility**: Adapt workflows to changing requirements.

Manual override is an essential feature for ensuring the reliability and adaptability of the JDHP publishing house.
