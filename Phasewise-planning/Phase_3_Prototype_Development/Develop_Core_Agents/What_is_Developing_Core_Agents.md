# What is Developing Core Agents?

Developing core agents involves building the foundational agents that perform the primary tasks in the system. These agents are critical for the basic functionality of the JDHP publishing house.

## Key Objectives:
1. Build the **Topic Selection Agent** to identify trending topics.
2. Develop the **Research Agent** to generate research reports.
3. Implement the **Publishing Agent** to manage report publishing and sales.

Core agents form the backbone of the JDHP system and must be robust and efficient.
