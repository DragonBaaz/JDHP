# What is Kafka?

Kafka is a distributed event streaming platform designed for high-throughput, fault-tolerant communication. It is widely used for real-time data pipelines and distributed systems.

## Key Features:
1. **Event Streaming**: Publish and subscribe to streams of records.
2. **Scalability**: Handle large volumes of data across distributed systems.
3. **Durability**: Persist messages for reliable delivery.

Kafka is an excellent choice for managing real-time communication between agents in the JDHP publishing house.
