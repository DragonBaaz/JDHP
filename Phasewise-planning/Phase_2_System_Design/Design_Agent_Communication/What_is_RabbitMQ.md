# What is RabbitMQ?

RabbitMQ is an open-source message broker that facilitates communication between distributed systems. It enables agents to exchange messages asynchronously, ensuring reliable and scalable communication.

## Key Features:
1. **Message Queues**: Store and forward messages between agents.
2. **Asynchronous Communication**: Decouple agents for better scalability.
3. **Reliability**: Ensure message delivery even in case of failures.

RabbitMQ is a powerful tool for managing agent communication in the JDHP publishing house.
